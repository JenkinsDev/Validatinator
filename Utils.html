(static) convertFieldValidationsToArray(validationInfo) → {Object}

Loops through each form and field and converts the field's string of validations to an array of field validations. 
  e.g:
  convertFieldValidationsToArray({
    "aForm": {
      "bField": "required|min:5|max:10"
    }
  }) =>
  {
    "aForm": {
      "bField": ["required", "min:5", "max:10"]
    }
